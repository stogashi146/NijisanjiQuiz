# NijisanjiQuiz
iOS用クイズアプリ
https://apps.apple.com/app/id1587645647
# 機能一覧
## タイトル画面
- Startボタン
## 難易度選択画面
- 「ふつう」「むずかしい」
- 「むずかしい」は「ふつう」を３０問以上正解で解禁

## クイズ画面
- 問題文、４択ボタン、残りタイム、ライフゲージ
- ３問不正解又は、時間切れが３回で失敗

# 使用技術
- Swift .4.2
- Xcode 12.5.1 
- StoryBoard
